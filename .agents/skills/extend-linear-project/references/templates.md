## 项目状态更新模板

### 范围内增强

```markdown
# 项目状态更新｜[YYYY-MM-DD]

**健康状态**：正常
**一句话摘要**：已将新增需求"[名称]"纳入项目范围，当前不影响主里程碑和上线计划。

## 本次完成
- 更新项目说明，补充新增需求的用户、场景、范围和验收标准。
- 新建 [n] 个 Issues。
- 规划当前/下一 Cycle 的纳入 Issue 和顺延项（如适用）。
- 建立 [n] 条依赖关系。

## 影响评估
- 范围：小幅增加
- 架构：无重大变更 / 有轻微调整
- 时间：暂不影响
- 风险：可控

## 下一步
- 完成新增 Issue 的详细拆分。
- 验证相关接口、数据和权限假设。
```

## 本次完成

- 新增/调整 Milestone：[名称]
- Cycle 调整：[当前/下一 Cycle 纳入与顺延说明]
- 新建 Issues：[列表]
