# Evidence Based Reporting

## 目的

基于 Linear、GitHub、本地和 web 事实生成项目报告。

## 报告内容

- 本周期完成。
- 进行中。
- 风险和阻塞。
- 决策与待确认项。
- GitHub PR/Actions 证据。
- 本地未提交变更提醒。
- 下一步建议。

## 要求

报告中不得出现无法追溯来源的事实陈述。无法确认的内容必须标为假设或待确认。
